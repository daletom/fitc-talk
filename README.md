# fitc-talk
Presentation during the [2021 FITC Web Unleashed](https://fitc.ca/event/webu21/), discussing how to improve pagespeed and supercharge your images.  Use a vanilla javascript live coding tutorial to show a poor performing website on Google Pagespeed Insights and step by step improvements to the score of a 100!
